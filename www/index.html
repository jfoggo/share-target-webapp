ShareTarget

This WebApp tests the "share_target" functionality. (Link to Github)

  1. You need to install this website as a WebApp via Mobile Chrome Browser on Android Device.
  2. Then you can open youtube app (as an example) and click on "share video" button.
  3. Here you need choose "ShareTarget" WebApp from the list of available apps.
  4. All shared information will be displayed below.
